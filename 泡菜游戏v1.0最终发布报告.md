## 🚀 使用指南

### 安装步骤
1. 将 `PickleGames-1.0.jar` 放入服务器的 `plugins` 文件夹
2. 启动服务器，插件会自动生成配置文件
3. 编辑 `config_zh_CN.yml` 自定义设置
4. 使用 `/pg reload` 重新加载配置

### 创建竞技场
1. 使用 `/pg gui` 打开管理界面
2. 点击"创建新竞技场"
3. 选择游戏类型
4. 按照屏幕提示设置坐标点
5. 使用 `/pg save` 保存竞技场

### 玩家加入游戏
- 命令: `/pg join <游戏名>`
- 告示牌: 创建 `[JOIN]` 告示牌
- 自动加入: 某些游戏支持靠近自动加入（如台球、足球）

## 🎉 特别功能

### 1. 台球常驻模式
- 配置 `proximity-joining: true`
- 玩家靠近台球桌自动加入
- 支持多桌同时运行
- 完美适合主城装饰

### 2. 足球团队模式
- 自动分队（蓝队 vs 红队）
- 支持观战模式
- 可自定义球的大小和速度
- 加速跳跃技能

### 3. 排行榜系统
- 全局排行榜（跨竞技场）
- 本地排行榜（单竞技场）
- 支持 MySQL 数据库
- 可设置记录过期时间

### 4. 经济系统集成
- 支持 Vault 经济
- 可设置游戏费用
- 可设置胜利奖励
- 支持赌注模式
