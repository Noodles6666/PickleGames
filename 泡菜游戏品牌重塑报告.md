## 🎮 命令系统更新

### 主命令
| 旧命令 | 新命令 | 别名 |
|--------|--------|------|
| `/lobbygames` | `/picklegames` | `/pg`, `/泡菜`, `/pickle`, `/游戏`, `/games`, `/lg` (兼容) |
| `/lgjoin` | `/pgjoin` | `/加入游戏`, 各游戏中文名, 各游戏英文名, `/lgjoin` (兼容) |
| `/lgjoinplayer` | `/pgjoinplayer` | `/pgforcejoin`, `/lgjoinplayer` (兼容) |

### 子命令
所有子命令保持不变，但命令前缀从 `/lg` 变为 `/pg`:
- `/pg join <游戏> [ID]` - 加入游戏
- `/pg quit` - 退出游戏
- `/pg restart` - 重新开始
- `/pg list [游戏]` - 列出竞技场
- `/pg tp <游戏> <ID>` - 传送到竞技场
- `/pg kick <玩家>` - 踢出玩家
- `/pg create <游戏>` - 创建竞技场
- `/pg delete <游戏> <ID>` - 删除竞技场
- `/pg gui` - 打开管理界面 ✨
- `/pg leaderboard` - 排行榜管理
- `/pg reload` - 重新加载配置

## 🚀 使用指南

### 安装
1. 将 `PickleGames-2.3.0.jar` 放入服务器的 `plugins` 文件夹
2. 重启服务器或使用 `/reload confirm`
3. 插件会自动生成 `config_zh_CN.yml` 配置文件

### 首次使用
1. 使用 `/pg gui` 打开管理界面
2. 点击"创建新竞技场"选择游戏类型
3. 按照聊天框提示设置坐标点
4. 使用 `/pg save` 保存竞技场
5. 使用 `/pg join <游戏>` 测试游戏

### 玩家命令
- `/pg join <游戏>` - 加入游戏
- `/pg quit` - 退出游戏
- `/pg list` - 查看可用游戏
- 或使用中文别名：`/台球`, `/足球`, `/扫雷` 等

### 管理员命令
- `/pg gui` - 打开管理界面 (推荐)
- `/pg create <游戏>` - 创建竞技场
- `/pg delete <游戏> <ID>` - 删除竞技场
- `/pg leaderboard` - 管理排行榜
- `/pg reload` - 重新加载配置
