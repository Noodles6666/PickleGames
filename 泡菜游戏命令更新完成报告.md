## 使用说明

### 主命令
- `/pg` 或 `/picklegames` - 主命令
- `/pg gui` - 打开管理界面（需要 picklegames.admin 权限）

### 管理界面功能
1. **创建新竞技场** - 选择游戏类型并按照指南创建
2. **管理现有竞技场** - 查看、编辑或删除竞技场
3. **排行榜管理** - 使用 `/pg leaderboard` 命令管理

### 创建竞技场流程
1. 使用 `/pg gui` 打开管理界面
2. 点击"创建新竞技场"
3. 选择要创建的游戏类型
4. 按照屏幕上的详细指南操作：
   - 使用 `/pg set` 设置坐标点
   - 使用 `/pg save` 保存竞技场
   - 使用 `/pg quit` 退出编辑模式
